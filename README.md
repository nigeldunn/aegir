# Aegir

This game is a space combat game focusing on dogfighting. Heavily inspired by
EvE, Descent, and the Wing Commander series (especially Privateer). The game is written in
Rust (https://www.rust-lang.org/) and uses the Bevy game engine (https://bevyengine.org/).

The final game is unlikely be open source. This is currently public as it is the very start of the project 
and is convenient for asking questions and getting code reviewed when necessary.
